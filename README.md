# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) in Rust.

## Usage

```bash
# Run a specific day (e.g., day 1)
cargo run -- 1

# Run with release optimizations
cargo run --release -- 1

# Run tests for a specific day
cargo test day01

# Run all tests
cargo test
```

## Project Structure

```
AoC-2025/
├── Cargo.toml
├── README.md
├── src/

│       ├── day02.rs
│       └── ...

    └── ...
```

## Adding Your Input

1. Go to [adventofcode.com](https://adventofcode.com/2025)
2. Log in and get your puzzle input for each day
3. Paste the input into the corresponding `inputs/dayXX.txt` file

> **Note:** AoC asks participants not to share their inputs publicly. Consider adding `inputs/` to your `.gitignore` if you plan to share your repository.
